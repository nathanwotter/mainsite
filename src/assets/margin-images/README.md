Store decorative margin images in this folder.

Accepted formats:
- `.avif`
- `.gif`
- `.jpeg`
- `.jpg`
- `.png`
- `.svg`
- `.webp`

How it works:
- The site picks one random image from this folder on each page load.
- That same image is shown in the left and right page margins.
- The image repeats vertically to fill the page height.
- Margin images only appear on large screens.

Tips:
- Wide images work best, since the left margin shows the left edge and the right margin shows the right edge.
- If you add or remove files here, rebuild or restart the Astro site so the new image list is included.
